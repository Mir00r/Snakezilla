# Audio Assets

Place your audio files here:

- `eat.mp3` – Sound effect when the snake eats food
- `game_over.mp3` – Sound effect on game over
- `tap.mp3` – UI button tap sound
- `bgm.mp3` – Background music loop (loopable track)

You can find free game audio at:
- [OpenGameArt.org](https://opengameart.org/)
- [Freesound.org](https://freesound.org/)
- [Kenney.nl](https://kenney.nl/assets?q=audio)
